什么是Node.js？

简单的说 Node.js 就是运行在服务端的 JavaScript。

Node.js是一个基于 Chrome V8 引擎的 JavaScript 运行环境；

Node.js使用一个事件驱动、非阻塞式 I/O 的模型，使其轻量且高效；

Node.js的软件包生态系统npm是全球最大的开源库生态系统。
